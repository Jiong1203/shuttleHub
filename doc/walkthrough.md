# ShuttleHub - 開發成果展示

本專案「ShuttleHub」已完成核心功能開發，包含獨立首頁、活動開團、報名系統、管理介面以及場地整合功能。

## 1. 專案概覽

使用 **Vue 3 + Vite** 建置，搭配 **Pinia** 進行狀態管理，並使用 **Vanilla CSS** 實現響應式與現代化設計。

### 核心功能

- **首頁 (Landing Page)**: 吸引人的 Hero 區塊與功能介紹，引導使用者開始使用。
- **活動瀏覽**: 查看所有羽球活動。
- **開團功能**: 建立新的羽球活動 (設定時間、地點、程度、費用)。
- **報名系統**: 使用者可報名參加活動。
- **管理介面**: 主辦者可管理自己的活動與審核報名。
- **場地整合**: 內建高雄市熱門羽球館清單，開團時可直接選擇。

## 2. 功能展示

### 🏠 全新首頁

獨立的 Landing Page，包含：

- **Hero Section**: 明確的價值主張與行動呼籲 (CTA)。
- **功能亮點**: 快速了解平台優勢 (快速開團、報名管理、尋找球友)。

### 📋 活動列表

展示所有即將到來的活動，卡片式設計清晰呈現關鍵資訊 (程度、費用、時間、地點)。

### 🏸 開新羽球團 (整合場地資料)

提供完整的表單，讓主辦者設定活動細節。

- **地點選擇**: 內建高雄市 40+ 間羽球館資料，可直接從下拉選單選擇，省去輸入地址的麻煩。
- **程度分級**: 支援新手友善、中級程度等分級。
- **自動驗證**: 確保所有必填欄位皆已填寫。

### 📝 活動詳情與報名

- 顯示活動完整描述與目前報名狀況 (人數/上限)。
- 使用者點擊「立即報名」即可加入。
- 若已額滿或已報名，按鈕狀態會自動更新。

### 👥 報名管理 (主辦者專用)

- 查看所有報名者列表。
- 審核功能：可「核准」或「拒絕」報名申請。
- 統計數據：即時顯示總報名人數與已核准人數。

## 3. 技術亮點

- **響應式設計**: 支援手機與桌面版面。
- **狀態管理**: 使用 Pinia 集中管理活動與報名資料，操作流暢無延遲。
- **元件化開發**: 重複使用 Button, Card, Input 等元件，保持介面一致性。
- **路由規劃**: 獨立首頁與活動列表路由，提升使用者體驗。
- **資料整合**: 將外部場地資料整合至前端應用，提升開團效率。
